## CartPole RL Learning + Implementation

This repo consists of a learning step-by-step implementations of the CartPole reinforcement learning using the Gymnasium, Stable-Baselines3 libraries. 

The phase*.py is how ste-by-step I built my intuition while reading various resources. The resources which I used to gain further knowledge are:

1. Optimal Control Course (EE6412) at IITM
2. [Gymnasium library docs](https://gymnasium.farama.org/api/env/)
3. PPO blog at [GFG](https://www.geeksforgeeks.org/machine-learning/a-brief-introduction-to-proximal-policy-optimization/)
4. [Actor critic concepts](https://www.geeksforgeeks.org/machine-learning/actor-critic-algorithm-in-reinforcement-learning/)
5. [Explanattion of PPO Algorithm](https://www.youtube.com/watch?v=5P7I-xPq8u8)
6. [SB3 docs](https://stable-baselines3.readthedocs.io/en/master/modules/ppo.html)
7. [RL blog - viewable as an HTML file](rl_blogs.html)
